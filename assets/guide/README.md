# 使い方ガイド用 画像・GIF 置き場

page2.html の大「？」モーダル（GUIDE_STEPS）で使う実素材を、このフォルダに置きます。
**ファイル名は下記のとおりにしてアップロードしてください**（page2.html 側がこの名前で参照します）。
すべて **正方形にトリミング** 前提です。ファイルが無い間は、コード側が自動でプレースホルダー表示に戻ります。

## ステップ2：NFC / QR の動作GIF
| ファイル名 | 内容 |
|---|---|
| `nfc-demo.gif` | スマホをNFCタグにかざして → URLが開くまでの一連の動き |
| `qr-demo.gif`  | カメラでQRコードを読み取って → URLが開くまでの一連の動き |

## ステップ3：土台カラー比較（複数色が写った「まとめ写真」2枚）
色ごとに切り抜かず、複数色が1枚に並んだ写真をそのまま表示する。

| ファイル名 | 内容 |
|---|---|
| `color-plain-group.jpg` | 無地5色（白・黒・青・黄・オレンジ）が並んだ写真 |
| `color-print-group.jpg` | 同じ絵を印刷した5色が並んだ写真 |

※ レッド・グリーンはこのまとめ写真に未収録。将来7色版に撮り直す／追加する場合は、
　この2枚を差し替えるだけでよい（page2.html のコメントにも明記）。

## ステップ4：サイズ比較（3サイズが写った「まとめ写真」1枚）
同じ白色・同じ印刷を、小・中・大で並べた写真をそのまま表示する。

| ファイル名 | 内容 |
|---|---|
| `size-group.jpg` | 白×同一印刷を 小・中・大 で並べた写真 |

---

差し替え時の対応箇所（page2.html 内・すべて「データ側」だけで完結）:
- GIF … `GIF_NFC_SRC` / `GIF_QR_SRC`
- 土台カラー … ステップ3の `gdPhoto('assets/guide/color-plain-group.jpg', …)` / `color-print-group.jpg`
- サイズ … ステップ4の `gdPhoto('assets/guide/size-group.jpg', …)`

写真の縦横比は固定していない（`max-width:100%` / `height:auto`）ので、縦長・横長どちらでもそのまま収まる。

※ 元素材（`color .HEIC` / `color色あり.HEIC` / `size.HEIC` / ルートの `NFC.gif` / `QR.gif`）は
　`.gitignore` でリポジトリ対象外。表示用に変換・縮小した版だけをこのフォルダに置いている。
